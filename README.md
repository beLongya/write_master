# 个人字迹仿写照片生成器

这是一个本地网页工作台，包含 OCR 识别、个人字库管理、缺字补录、手写照片生成和 Word/PDF 文档转手写骨架。

## 项目初衷与停止原因

我最初做这个项目，是因为学校作业里需要抄写的内容太多，所以想尝试做一个程序来辅助生成手写效果，减少机械重复的抄写负担。

后来实际做下来发现，仿真手写照片的效果并不理想：字迹、纸面、排版和真实书写之间仍然有明显差距，继续投入也不一定能达到可用效果。因此这个项目暂时停止维护，仅作为一次技术尝试和原型记录保留。

请不要将本项目用于考试、作业提交、签名、证明、合同或其他可能造成误导的场景。

## 启动

1. 如果要使用百度手写 OCR，先配置环境变量：

   ```powershell
   $env:BAIDU_OCR_API_KEY="你的 API Key"
   $env:BAIDU_OCR_SECRET_KEY="你的 Secret Key"
   ```

2. 启动本地服务：

   ```powershell
   cd C:\Users\Lenovo\Desktop\ai\write
   & 'C:\Users\Lenovo\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe' 'C:\Users\Lenovo\Desktop\ai\write\server.py'
   ```

3. 打开浏览器访问：

   ```text
   http://127.0.0.1:8787
   ```

## 功能

- 上传整张手写照片，调用百度 OCR 识别文字。
- 校对 OCR 文本后，按文字块自动切分为单字候选。
- 候选字样可修改、删除，并入库到当前字库风格。
- 支持工整版、日常版、练习版等多风格字库。
- 显示已收录字符数、字样数量、常用字覆盖率和当前文本缺字。
- 自动生成缺字补录任务，可复制后手写拍照继续完善字库。
- 支持作文本、横线纸、方格纸、A4 白纸。
- 支持上传自定义纸张图片，并在预览中框选文字书写区域。
- 支持黑色中性笔、蓝色圆珠笔、钢笔风格。
- 支持上传 `.docx` / `.pdf`，提取纯文本后按页生成手写图片。

## 说明

- `.docx` 文本提取使用 Python 标准库处理。
- `.pdf` 文本提取会优先使用当前 Python 环境中的 `pypdf` 或 `PyPDF2`；如果没有安装，会在页面提示。
- 第一版文档转换只提取文本，不保留复杂版式。
- 照片会发送给百度 OCR 服务识别；入库后的字样保存在浏览器本地。

## 生成效果调节

- 默认字号已调小，纸张在画面中占比更大，生成结果会更接近真实整张纸。
- 选择“自定义纸张”并上传图片后，会把该图片作为纸面背景。
- 点击“框选书写区域”，可在预览纸张上拖拽限定文字写入范围；再次点击可退出框选。
- 点击“重置区域”可恢复默认纸面书写范围。
